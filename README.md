# Jukkai

Bun + Turborepo workspace for the clean Jukkai product repo.

## Commands

```sh
bun install
bun run dev:marketing
bun run check
bun run build
```

The marketing site lives in `apps/marketing` and runs on Astro with Vite, CSS
Modules, Astro type checking, ESLint, Stylelint, Prettier, Vitest, Husky, and
lint-staged.
